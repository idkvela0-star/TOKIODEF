Cual es la diferencia entre Gif, Html y Jpeg?

R//: GIF y JPEG son formatos de imagen: el GIF admite animaciones y pocos colores, mientras que el JPEG sirve para fotos con muchos colores pero sin animación ni transparencia. En cambio, HTML es un lenguaje de código que estructura páginas web y puede incluir imágenes como GIF o JPEG.
